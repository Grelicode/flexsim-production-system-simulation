# flexsim-production-system-simulation
Discrete - event simulation of a smart factory system in FlexSim including production line, robots, ASRS warehouse and material flow optimization.

# Overview
This project presents a smart factory simulation model developed in FlexSim. The system combines a production line, industrial robots, operators, and a warehouse (ASRS + forklift transport) to simulate real industrial material flow. The goal of the project was to analyze system performance, identify bottlenecks, and evaluate automation strategies.
